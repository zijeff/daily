# 基于 pthread 的多线程数独合法性检测

## 项目简介

本项目使用 **C++ + pthread** 实现了一个基于多线程的数独合法性检测程序。  
程序用于判断一个 9×9 数独矩阵是否满足数独规则，即：

- 每一行都包含 1~9，且不重复
- 每一列都包含 1~9，且不重复
- 每一个 3×3 子宫格都包含 1~9，且不重复

本程序采用多线程方式并发完成上述检查任务，这也是《操作系统概念》中关于线程编程的经典练习之一。

## 题目背景

在操作系统课程中，多线程的核心思想之一是：  
**将一个可以拆分的任务划分为多个相互独立的子任务，并交由多个线程并发执行。**

数独合法性检测正好适合这种方式，因为可以将整体判断拆分为以下 11 个独立任务：

- 1 个线程检查所有行
- 1 个线程检查所有列
- 9 个线程分别检查 9 个 3×3 子宫格

主线程在等待所有子线程执行完毕后，对各线程结果进行汇总，最终判断该数独是否合法。

## 功能说明

本程序实现了以下功能：

- 检查数独每一行是否合法
- 检查数独每一列是否合法
- 检查每个 3×3 子区域是否合法
- 使用 `pthread_create()` 创建线程
- 使用 `pthread_join()` 等待所有线程结束
- 汇总所有线程执行结果并输出最终判断结果

## 线程分工

程序总共创建 **11 个线程**，分工如下：

| 线程编号 | 功能 |
|---------|------|
| `tid[0]` | 检查所有行 |
| `tid[1]` | 检查所有列 |
| `tid[2] ~ tid[10]` | 分别检查 9 个 3×3 子宫格 |

结果保存在全局数组 `res[11]` 中：

- `res[0]`：行检查结果
- `res[1]`：列检查结果
- `res[2] ~ res[10]`：9 个子宫格检查结果

若所有位置均为 `1`，则说明数独合法；否则不合法。
