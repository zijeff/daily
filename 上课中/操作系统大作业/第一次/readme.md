#### 功能描述
如注释中所说，该模块实现了生成 0-100 内的所有质数。因为数据范围较小，我们采用了**试除法**直接求解。
如果数据范围更大，我们考虑使用更高效的算法，例如埃氏筛或者欧拉筛。
#### 部署方法

部署使用 **Makefile** ，具体文件已经编写好。只需要在终端界面输入：

```bash
make
```

将会生成内核模块文件，接着输入，

```bash
sudo insmod main.ko
```

使用如下命令查看我们编写的模块有没有被导入，

```bash
lsmod | grep main
```

因为模块内部并没有加入交互逻辑，所以直接卸除模块，

```bash
sudo rmmod main
```

最后，只需查看内核文件即可找到模块程序的输出信息。

```bash
sudo dmesg
```
